# SourcePort

```json title="Multi"
{
    "Type": "SourcePort",
    "Parameter": [
        10000,
        10001
    ]
}
```

```json title="Single"
{
    "Type": "SourcePort",
    "Parameter": 10000
}
```

SourceIP matches port of the client.

The parameter is a listable of port numbers, ranging from 0 to 65535.
