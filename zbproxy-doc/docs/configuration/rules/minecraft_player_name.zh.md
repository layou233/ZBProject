# MinecraftPlayerName

```json
{
    "Type": "MinecraftPlayerName",
    "Parameter": [
        "la_you",
        "AnyPlayer_",
        "list:allow_player"
    ]
}
```

!!! note
    你应该先嗅探 Minecraft 协议。

MinecraftPlayerName 按原样匹配 Minecraft 玩家名。

这是大小写敏感的。

你可以通过 "lists:" 前缀和 List Tag 来引用 Lists 中的列表。
