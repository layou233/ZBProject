# 逻辑规则

```json title="与"
{
    "Type": "and",
    "Parameter": [
        {
            "Type": "",
            // ...
        },
        {
            "Type": "",
            // ...
        },
        // ...
    ]
}
```

```json title="或"
{
    "Type": "or",
    "Parameter": [
        {
            "Type": "",
            // ...
        },
        {
            "Type": "",
            // ...
        },
        // ...
    ]
}
```

逻辑规则包含两个或更多子规则，并且为了取得逻辑运算结果匹配他们。

当逻辑结果是真时，该规则被匹配。

可用的逻辑规则类型是： `and` 和 `or`。全部是小写字母。

子规则即使命中了也不会执行任何嗅探或重写。
它们的匹配只是为了一个结果。
